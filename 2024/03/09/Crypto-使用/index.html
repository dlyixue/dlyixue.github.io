Crypto++部署

本文采用Windows子系统wsl2进行部署测试

  • 从官网下载Crypto++ (https://www.cryptopp.com/)
  • make -j4
  • make install PREFIX=./path 使用path指定安装路径
  • g++ hw1.cpp -I /usr/local/bin/include/cryptopp -o hw1.exe /usr/local/bin/lib/libcryptopp.a 使用g++进行编译
